Setup Guide

Prerequisites

  • A modern web browser (Chrome, Firefox, Safari, or Edge)
  • Text editor (VS Code, Sublime Text, etc.)
  • Basic knowledge of HTML, CSS, and JavaScript

Installation Steps

Step 1: Clone the Repository

Clone the project repository to your local machine:

git clone https://github.com/hvbombay/project.git

Step 2: Navigate to Project Directory

Open the project folder in your text editor

Step 3: Open HTML Files

Open any HTML file directly in your browser to view the components

Note: No server setup is required for basic functionality. Simply open the HTML files in your browser.

Troubleshooting

If you encounter any issues, please check:

  • All files are in the correct directory
  • File paths in HTML are correct
  • Browser console for errors
